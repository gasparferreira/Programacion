# Semana 1 - Ejercicios en C | Programacion 1

Este repositorio muestra los ejercicios realizados durante la primera semana de programacion 1. 

---

## Ejercicio 1 - Presentacion personal

### ¿Que hace?
El programa solicita al usuario los siguientes datos: nombre, apellido, ciudad de origen y edad. Luego muestra un mensaje con esa informacion.

### ¿Cómo se ejecuta?
Abrir el archivo correspondiente en Visual Studio Code y presionar el botón **Run** para ejecutar el programa.
