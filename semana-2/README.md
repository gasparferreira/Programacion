## Ejercicio 6 - Datos personales

### ¿Que hace?
El programa pide al usuario la inicial de su nombre, su altura, el año actual y su año de nacimiento, mostrando el valor ASCII de la letra ingresada, luego evalúa si la altura está dentro de un rango promedio y calcula su edad.

### ¿Cómo se ejecuta?
Abrir el archivo correspondiente en Visual Studio Code y presionar el botón **Run** para ejecutar el programa.

## Ejercicio 14 - El código secreto

### ¿Que hace?

El programa pide una letra mayúscula y verifica que esté entre ‘A’ y ‘Z’; luego muestra su “código secreto” desplazándola 3 posiciones en el abecedario.
Si la letra es ‘Z’, el desplazamiento vuelve al inicio del abecedario, mostrando ‘C’.

### ¿Cómo se ejecuta?
Abrir el archivo correspondiente en Visual Studio Code y presionar el botón **Run** para ejecutar el programa.
